## 🚀 Getting Started

### Prerequisites
- Python 3.9 or higher
- pip

### Installation

1. Clone the repository:
```bash
git clone https://github.com/raniarashid780-sketch/python-practice.git
cd python-practice
```

2. Install the required packages:
```bash
pip install -r requirements.txt
pip install numpy
```

3. Run a lesson file:
```bash
python 01_variables_and_print.py
python 05_functions.py
python 09_numpy_basics.py
```
