## 🚀 Getting Started

### Prerequisites
- Python 3.9 or higher
- pip (Python package manager)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/raniarashid780-sketch/python-practice.git
cd python-practice
```

2. Install dependencies (if any):
```bash
pip install -r requirements.txt
```

3. Run any file:
```bash
python 01_variables_and_print.py
python 02_data_types.py
```
